## Abstract

This study examined whether augmenting CNNs with synthetic images generated by GANs significantly improves image classification performance across three benchmark datasets of varying complexity: MNIST, CIFAR-10, and Tiny ImageNet. A total of 630 independent experimental runs (105 per condition across three datasets) were conducted using a DCGAN architecture for synthetic image generation and a standardized CNN for classification. Results indicated that unconditional GAN augmentation did not improve classification performance on any dataset:

- **MNIST:** Small but significant decrease (*d* = 1.18, *p* < .001, negative direction)
- **CIFAR-10:** No significant effect (*d* = 0.03, *p* = .840)
- **Tiny ImageNet:** Substantial degradation (*d* = 2.11, *p* < .001, negative direction)

## Experimental Design

| Parameter | Value |
|-----------|-------|
| Design | Quantitative experimental-causal comparative |
| Sample size | n = 105 per condition (G*Power: power = 0.95, α = .05, d = 0.5) |
| Total runs | 630 (3 datasets × 2 conditions × 105 runs) |
| GAN architecture | Deep Convolutional GAN (DCGAN), unconditional |
| CNN architecture | Standardized CNN per dataset complexity |
| Platform | RunPod GPU cloud (NVIDIA A40/A100) |

## Results Summary

| Dataset | CNN Accuracy | CNN+GAN Accuracy | Cohen's *d* | *p* value | Test |
|---------|-------------|-------------------|-------------|-----------|------|
| MNIST | 99.58% | 99.51% | 1.18 (negative) | < .001 | Independent *t*-test |
| CIFAR-10 | 90.73% | 90.74% | 0.03 (negligible) | .840 | Independent *t*-test |
| Tiny ImageNet | 42.67% | 38.21% | 2.11 (negative) | < .001 | Welch's *t*-test |

## Datasets

This research uses three publicly available benchmark datasets:

- **MNIST** — LeCun, Y., Bottou, L., Bengio, Y., & Haffner, P. (1998). Gradient-based learning applied to document recognition. *Proceedings of the IEEE, 86*(11), 2278–2324. https://doi.org/10.1109/5.726791
- **CIFAR-10** — Krizhevsky, A. (2009). *Learning multiple layers of features from tiny images* [Technical report]. University of Toronto.
- **Tiny ImageNet** — Le, Y., & Yang, X. (2015). *Tiny ImageNet visual recognition challenge*. Stanford University CS 231N Course Project.
